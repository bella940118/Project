
Project Name: Event Planner


Description:
This project allows users to manage and attend events efficiently. Users can choose to interact
as a Host (who creates and manages events) or an Attendee (who can join events and check for schedule conflicts)

Files in the Project:
main.py: Contains the core `Event` and `EventManager` classes to handle event details and attendees.
test.py: test function in main.py
event.py: Uses `main.py` to manage event scheduling and event-related operations.
event_manager.py: Implements the EventManager class, responsible for adding, removing, listing, and checking event
details. It also handles time conflict checks and event file loading.
text1.txt: A text file storing preloaded events'


How to Run the Project:
Run event_manager.py
